A Sonar Scanner With Alert System : 

Components Used –  
Arduino Uno 
A Ultrasonic Sensor 
A Servo 
A buzzer 
A RGB LED 
3 Resistors 
Jumper Wires  

Code Logic – The ultrasonic sensor sends a sound wave 
and determines the distance between the object if 
detected and the initial idea of the project was to 
attach the ultrasonic sensor on top of the servo and 
make It such that the servo rotates and covers a full 180 
degree area and the ultrasonic sensor detects any 
objects in that 180 degree and if the ultrasonic sensor 
does detect anything then with the distance the servo 
behaviour changes i.e. if the distance is above 20 then 
servo rotates but a safe/caution message is displayed in 
serial monitor with respect to the distance being 
greater than 50 or between 20-50 respectively and if 
the distance is less than 20 then the servo stops until 
the object detected is removed. But due to Lack of 
working servo ,the product couldn’t be done in 
hardware and therefore a similar one is built in 
Tinkercad and the although the servo and ultrasonic isnt attached ,
the servo stops when a object is detected by a ultrasonic sensor at a
distance less than 20.
