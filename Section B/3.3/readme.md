Automatic Night Lamp :

Components Used –  
Arduino Uno 
A LDR 
A LED 
2 Resistors  
Jumper wires 

Code Logic – Using the LDR(Light Dependent Resistor), 
the ambient light level is detected with the use of 
analogRead() and using map() function the LED’s 
brightness is controlled by the LDR and as light falling 
on LDR increase , the brightness on LED decreases and 
vice versa and then the light level is printer in serial 
monitor.
