Morse Code Transmitter :

Components Used –  
Arduino Uno 

A Buzzer 

2 LED’s 

2 Resistors 

Jumper wires


Code Logic – An array is created for both alphabets and 
numbers with their morse code stored respectively in 
order and with the help of Serial monitor, user input is 
read and reads each character and checks if the 
character belongs to alphabets or numbers and 
accordingly the array’s index Is chosen and then the 
morse code is represented with both the buzzer and 
RED LED  and also a BLUE LED is connected which glows 
when there’s a sentence with space between words .
